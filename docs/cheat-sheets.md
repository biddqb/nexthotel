# Bảng tra cứu nhanh 1 trang

In trang này ra giấy, dán gần máy tính.

---

## ☁️ LỄ TÂN — CHECKLIST 1 PHÚT

### Đầu ca
```
1. Đăng nhập  → http://192.168.x.x:8080
2. Mở ca      → thanh trái, nút "Mở ca"
3. Xem Lịch   → chú ý "Khách nhận phòng hôm nay"
```

### Khách đến bất ngờ (walk-in)
```
1. Lịch  → bấm ô trống của phòng + ngày
2. Chọn / thêm nhanh khách
3. Điền cọc nếu có  →  Lưu
4. Bấm "Nhận phòng" →  giao chìa
```

### Khách đã đặt trước đến
```
1. Tìm trong sidebar "Khách nhận phòng hôm nay"
2. Bấm tên khách → khay mở
3. Bấm "Nhận phòng"
```

### Khách ăn minibar / giặt ủi
```
1. Mở đặt phòng của khách
2. Dịch vụ phát sinh  →  + Thêm dịch vụ
3. Loại + Mô tả + Số tiền  →  Lưu
```

### Khách trả phòng
```
1. Kiểm tra "Còn lại" = 0đ
2. Nếu còn → Ghi nhận thanh toán (có nút "Còn lại" để điền nhanh)
3. Bấm "Trả phòng"  →  phòng tự → Bẩn
4. In hoá đơn nếu khách cần
```

### Sau khi dọn phòng
```
1. Mở "Buồng phòng"
2. Bấm ô phòng vừa dọn
3. Bấm "Đánh dấu sạch" (nút xanh lá)
```

### Cuối ca
```
1. Bấm chấm xanh "Đang trong ca" ở thanh trái
2. Ghi chú bàn giao  →  Kết thúc ca
```

### Màu trạng thái phòng (Buồng phòng)

| Màu | Ý nghĩa |
|---|---|
| 🟢 Xanh lá | Sạch |
| 🟠 Cam | Bẩn |
| 🔵 Xanh dương | Đã kiểm tra |
| 🔴 Đỏ | Đang bảo trì |
| ⚫ Xám đậm | Ngưng sử dụng |

### Màu đặt phòng (Lịch)

| Màu | Ý nghĩa |
|---|---|
| Xám xanh | Đã xác nhận, chờ nhận phòng |
| 🟢 Xanh lá | Đang ở |
| ⚪ Xám | Đã trả phòng |
| 🔴 Đỏ gạch | Đã huỷ |

### Chấm thanh toán (góc phải thanh đặt phòng)

| Dấu | Ý nghĩa |
|---|---|
| ● (trắng đầy) | Đã thanh toán đủ |
| ○ (viền trắng) | Mới đặt cọc |
| (không có) | Chưa thanh toán |

## 🔧 QUẢN LÝ — CHECKLIST HÀNG NGÀY

### Sáng
```
□ Kiểm tra phòng sẵn sàng cho khách đến hôm nay (Buồng phòng)
□ Giải quyết vấn đề lễ tân báo từ ca đêm (ghi chú bàn giao)
□ Xem báo cáo tuần trước (nếu chưa)
```

### Trong ngày
```
□ Xử lý yêu cầu đặc biệt của khách (giảm giá, sửa dịch vụ nhầm)
□ Nhập chi phí phát sinh (Chi phí → Thêm)
□ Duyệt khách vào danh sách đen nếu có
```

### Cuối ngày
```
□ Cài đặt → Kết sổ cuối ngày (chọn ngày hôm qua)
□ Giải quyết hết cảnh báo vàng
□ Ghi chú → Chạy kết sổ
```

### Cuối tuần
```
□ Báo cáo → Tuần này → chụp màn hình lưu
□ Cài đặt → Sao lưu → Tạo bản sao lưu ngay
□ Copy file sao lưu ra USB / cloud cá nhân
```

## 👔 GIÁM ĐỐC — CHECKLIST HÀNG THÁNG

```
□ Xem Báo cáo → Tháng trước:
   - RevPAR so với tháng trước nữa

   - Danh mục nào tăng bất thường
□ Cài đặt → Lịch sử hoạt động:
   - Có sự kiện bất thường nào không
□ Cài đặt → Tài khoản:
   - Có ai nghỉ cần khoá tài khoản
□ Cài đặt → Sao lưu → Tạo bản sao lưu:
   - Copy ra ổ cứng ngoài
```

## 🔑 MẬT KHẨU & TÀI KHOẢN
